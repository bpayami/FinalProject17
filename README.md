# Missing from Magnet

Hi there, I see that you've stumbled across my game!  Please read on to find out more information & I hope you enjoy playing my game as much as I enjoyed making it!

# Game Synopsis
My game, entitled "Missing from Magnet," is an escape adventure type game.  The last thing the player can remember was stressing out over midterms and wakes up not being able to recognize their surroundings or remember how they got there.  The player must go from room to room, exploring the mansion and interacting with items in the rooms along the way.  In order to escape, the player must ultimately find a key and two escape codes ...but be wary, because there is danger along the way, too!
